# рҹӨҹ Sign Language Bridge вҖ” Multilingual Two-Way Communication System

> A real-time, bidirectional sign language communication system that bridges the gap between hearing-impaired individuals and non-signers using **MediaPipe**, **Machine Learning**, and **multilingual text-to-speech**.

---

## рҹ“Ң Overview

**Sign Language Bridge** enables natural, two-way communication by:

- рҹ“· Detecting hand gestures via webcam using **MediaPipe**
- рҹ§  Classifying gestures with a trained **ML model** (RandomForest / SVM / MLP)
- рҹ”Ҡ Converting recognized signs to **text and speech** in the user's language
- вңҚпёҸ Converting typed text back into **visual sign language** (video/image display)
- рҹҢҗ Supporting **multiple languages** вҖ” English, Hindi, Tamil, Telugu, French, and more

## вҡҷпёҸ Installation

### 1. Clone the repository

```bash

cd sign-language-bridge
```

### 2. Create a virtual environment


## рҹҡҖ Quick Start

### Without a trained model (rule-based fallback)

```bash
python main.py
```

The system will use a built-in rule-based gesture recognizer that detects:
`Hello`, `Stop`, `Yes`, `Good` from hand shape geometry.

---

## рҹҸӢпёҸ Training Your Own Gesture Model

### Step 1: Collect training data

```bash
python collect_data.py --label hello --samples 200
python collect_data.py --label stop --samples 200
python collect_data.py --label thank_you --samples 200
python collect_data.py --label please --samples 200
# ... repeat for each gesture
```

Press **SPACE** to start/stop recording during collection.

### Step 2: Train the classifier

```bash
python train.py                     # Default: RandomForest
python train.py --model svm         # Support Vector Machine
python train.py --model mlp         # Neural Network
```

### Step 3: Run with trained model

```bash
python main.py
```

## вҢЁпёҸ All CLI Options

| Argument        | Default    | Description                              |
|-----------------|------------|------------------------------------------|
| `--cam`         | `0`        | Webcam index                             |
| `--lang`        | `en`       | Output language code                     |
| `--mode`        | `both`     | `both`, `sign-to-text`, `text-to-sign`   |
| `--model`       | auto-found | Path to trained `.pkl` model             |
| `--no-speech`   | off        | Disable text-to-speech                   |
| `--confidence`  | `0.75`     | Minimum gesture confidence threshold     |
